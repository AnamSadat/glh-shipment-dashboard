# Vehicle Shipment Monitoring Dashboard

This system provides a web-based dashboard for monitoring vehicle shipment operations at PT Garis Laut Handal.
Features include:

- Real-time shipment tracking (manual status updates)
- Delivery timeline visualization
- Shipping rate lookup
- Demand forecasting using a Machine Learning model (Moving Average)
- Multi-role access (Admin & User)

Developed for operational use and academic research (Final Project).
